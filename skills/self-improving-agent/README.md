# Self-Improving Agent (Trae 优化版)

让 Trae AI 永久记忆你的编码习惯、偏好、规则和错误教训，越用越贴合你的风格。

## 📋 功能特性

- ✅ **自动静默学习** - 用户纠正后自动记忆，无需手动操作
- ✅ **错误历史记录** - 记录命令失败、代码报错，避免重复踩坑
- ✅ **编码规则记忆** - 记住你的代码风格、命名规范等
- ✅ **Trae 路径适配** - 使用 `.trae/learnings/` 目录，完全符合 Trae 规范
- ✅ **开箱即用** - 自动初始化，无需额外配置
- ✅ **优先级管理** - 用户手动规则 > 自动学习规则 > 默认规则

## 🚀 快速开始

### 1. 运行初始化脚本

在项目根目录打开 PowerShell 终端，执行：

```powershell
.\.trae\skills\self-improving-agent\init.ps1
```

### 2. 重启 Trae IDE

初始化完成后，重启 Trae IDE 使技能生效。

### 3. 开始使用

现在 Trae AI 会自动：
- 记住你的编码习惯
- 记录错误并避免重复
- 越来越贴合你的风格

## 📖 使用说明

### 学习记录是如何工作的？

1. **用户反馈**：当你纠正 AI 的输出时，AI 会自动记录
2. **错误记录**：当命令执行失败或代码报错时，AI 会自动记录
3. **规则应用**：下次执行任务时，AI 会先读取所有记录并应用

### 查看学习记录

你可以随时查看学习记录文件：

```powershell
# 查看规则记录
Get-Content .trae\learnings\LEARNINGS.md

# 查看错误记录
Get-Content .trae\learnings\ERRORS.md
```

### 手动添加规则

你也可以直接编辑 `.trae/learnings/LEARNINGS.md` 手动添加规则：

```markdown
- 2026-04-16 规则：所有注释必须使用中文
  来源：用户手动
  场景：所有文件
```

## ❓ 常见问题

### Q: 学习记录会同步吗？

A: 学习记录存储在项目的 `.trae/learnings/` 目录下，如果你使用 Git，可以提交这些文件来共享学习记录。

### Q: 如何重置学习记录？

A: 删除 `.trae/learnings/` 目录下的文件，然后重新运行 `init.ps1`。

### Q: 规则的优先级是怎样的？

A: 用户即时指令 > 用户手动规则 > 自动学习规则 > 默认规则。
