# 文档格式标准

本文档定义了启金供应链票据管理平台详细设计文档的标准格式规范。

## 1. 页面设置

### 1.1 标准参数

| 项目 | 标准值 | docx-js DXA值 |
|------|--------|---------------|
| 纸张大小 | A4 (210mm × 297mm) | width: 11906, height: 16838 |
| 方向 | 纵向 | 默认 |
| 上边距 | 2.54 cm | 1440 DXA |
| 下边距 | 2.54 cm | 1440 DXA |
| 左边距 | 3.17 cm | 1800 DXA |
| 右边距 | 3.17 cm | 1800 DXA |
| 页眉 | 1.5 cm | 850 DXA |
| 页脚 | 1.75 cm | 1000 DXA |

### 1.2 docx-js 页面设置代码

```javascript
const doc = new Document({
  sections: [{
    properties: {
      page: {
        size: { width: 11906, height: 16838 },
        margin: { top: 1440, right: 1800, bottom: 1440, left: 1800 }
      }
    }
  }]
});
```

### 2.3 正文字体

| 元素 | 字体 | 字号 | docx-js size | 行距 | 对齐 |
|------|------|------|--------------|------|------|
| 正文 | 宋体 (SimSun) | 小四号 (12pt) | 24 | 1.5倍行距 (360) | 两端对齐 |
| 段落首行缩进 | - | 2字符 | 480 DXA | - | - |
| 表格内容 | 宋体 (SimSun) | 五号 (10.5pt) | 21 | 单倍行距 (240) | 居中/两端 |
| 图注 | 宋体 (SimSun) | 五号 (10.5pt) | 21 | - | 居中 |
| 页码 | Times New Roman | - | 20 | - | 居中 |

### 2.4 特殊文字

| 类型 | 样式要求 | docx-js实现 |
|------|----------|-------------|
| 代码块 | Consolas, 10pt, 浅灰背景 | font: "Consolas", size: 20 |
| 关键字 | 黑体加粗 | font: "SimHei", bold: true |
| 强调 | 红色加粗 | color: "FF0000", bold: true |
| 注释 | 宋体六号，灰色 | size: 18, color: "808080" |

## 3. 表格样式

### 3.1 表格边框

| 位置 | 线型 | 线宽 | 颜色 | docx-js实现 |
|------|------|------|------|-------------|
| 外框 | 单线 | 1磅 | 黑色 | BorderStyle.SINGLE, size: 1 |
| 内框 | 单线 | 0.5磅 | 黑色 | BorderStyle.SINGLE, size: 1 |
| 表头边框 | 单线 | 1磅 | 黑色 | 同上 |

### 3.2 docx-js 表格边框定义

```javascript
const TABLE_BORDER = { 
  style: BorderStyle.SINGLE, 
  size: 1, 
  color: "000000" 
};
const TABLE_BORDERS = { 
  top: TABLE_BORDER, 
  bottom: TABLE_BORDER, 
  left: TABLE_BORDER, 
  right: TABLE_BORDER 
};
```

### 3.3 表格对齐

| 元素 | 水平对齐 | 垂直对齐 | docx-js实现 |
|------|----------|----------|-------------|
| 表头 | 居中 | 居中 | AlignmentType.CENTER, VerticalAlign.CENTER |
| 内容-文字 | 两端对齐 | 居中 | AlignmentType.JUSTIFIED, VerticalAlign.CENTER |
| 内容-数字 | 居右 | 居中 | AlignmentType.RIGHT, VerticalAlign.CENTER |
| 内容-代码 | 左对齐 | 顶端 | AlignmentType.LEFT, VerticalAlign.TOP |

### 3.4 表格单元格样式

```javascript
new TableCell({
  borders: TABLE_BORDERS,
  width: { size: 4680, type: WidthType.DXA },
  shading: { fill: "D5E8F0", type: ShadingType.CLEAR },
  margins: { top: 80, bottom: 80, left: 120, right: 120 },
  verticalAlign: VerticalAlign.CENTER,
  children: [new Paragraph({
    alignment: AlignmentType.CENTER,
    children: [new TextRun({ text: "内容", size: 21, font: "SimSun" })]
  })]
})
```

### 3.5 表格编号与标题

- 编号格式：`表 X-Y`
- 标题位置：表格上方居中
- 标题字体：宋体五号加粗
- 与表格间距：0.5行

## 4. 图表样式

### 4.1 流程图规范

```
┌─────────────┐
│   开始/结束  │  圆角矩形：开始/结束
├─────────────┤
│    处理      │  矩形：处理活动
├─────────────┤
│    判断      │  菱形：判断/分支
├─────────────┤
│     ↓       │  连接线带箭头
└─────────────┘
```

### 4.2 时序图规范

```
┌────────┐     ┌────────┐     ┌────────┐
│  Actor │     │ System │     │  3rd   │
└───┬────┘     └───┬────┘     └───┬────┘
    │              │              │
    │───请求──────>│              │
    │              │              │
    │<───响应──────│              │
    │              │              │
```

### 4.3 状态图规范

```
┌─────────┐
│  状态1  │──事件A──>┌─────────┐
└─────────┘         │  状态2  │
                    └─────────┘
       ▲               │
       │───────────────┘
          事件B/退出
```

### 4.4 图表编号与标题

- 编号格式：`图 X-Y`
- 标题位置：图表下方居中
- 标题字体：宋体五号加粗
- 与图表间距：0.5行
- 图表内文字：宋体六号

## 5. 章节编号规则

### 5.1 编号层级

```
第1章  系统概述
  1.1  业务背景
    1.1.1  业务起源
    1.1.2  业务现状
  1.2  设计目标
第2章  功能模块划分
  2.1  模块A
    2.1.1  功能1
    2.1.2  功能2
  2.2  模块B
```

### 5.2 编号深度

- 推荐最大深度：4级
- 超过4级使用项目符号

## 7. 页眉页脚

### 7.1 页眉设置

```javascript
headers: {
  default: new Header({
    children: [new Paragraph({
      alignment: AlignmentType.CENTER,
      children: [new TextRun({ 
        text: "BEMP详细设计文档", 
        size: 20, 
        font: "SimSun" 
      })]
    })]
  })
}
```

### 7.2 页脚设置

```javascript
footers: {
  default: new Footer({
    children: [new Paragraph({
      alignment: AlignmentType.CENTER,
      children: [
        new TextRun({ text: "第 ", size: 20, font: "SimSun" }),
        new TextRun({ children: [PageNumber.CURRENT], size: 20 }),
        new TextRun({ text: " 页", size: 20, font: "SimSun" })
      ]
    })]
  })
}
```
