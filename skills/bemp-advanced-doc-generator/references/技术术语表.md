# 技术术语表

本文档定义了 BEMP 业务功能详细设计文档中常用的技术术语。

## 一、票据业务术语

### 1.1 基础票据术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 商业汇票 | Commercial Draft | 出票人签发的，委托付款人在指定日期无条件支付确定金额给收款人或持票人的票据 |
| 银行承兑汇票 | Banker's Acceptance | 由银行作为承兑人的商业汇票 |
| 商业承兑汇票 | Trade Acceptance | 由银行以外的付款人承兑的商业汇票 |
| 电子商业汇票 | Electronic Commercial Draft | 依托电子商业汇票系统，以数据电文形式制作的商业汇票 |
| 出票 | Issue | 出票人签发票据并将其交付给收款人的票据行为 |
| 承兑 | Acceptance | 汇票付款人承诺在汇票到期日支付汇票金额的票据行为 |
| 背书 | Endorsement | 持票人为将票据权利转让给他人或者将一定的票据权利授予他人行使，而在票据背面或者粘单上记载有关事项并签章的行为 |
| 贴现 | Discount | 持票人在汇票到期日前，为了取得资金，贴付一定利息将票据权利转让给银行的票据行为 |
| 转贴现 | Rediscount | 金融机构为了取得资金，将未到期的已贴现商业汇票再以贴现方式向另一金融机构转让的票据行为 |
| 再贴现 | Re-discount | 中央银行通过买进商业银行持有的已贴现但尚未到期的商业汇票，向商业银行提供融资支持的行为 |

### 1.2 票据市场术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 上海票据交易所 | Shanghai Commercial Paper Exchange, SHCPE | 中国票据市场的核心基础设施，提供票据交易、登记托管、清算结算等服务 |
| 票交所 | PBOC/CCDC | 上海票据交易所的简称 |
| 票据交易 | Bill Trading | 在票据市场进行的票据买卖行为 |
| 登记托管 | Registration and Custody | 票交所对票据权利进行登记和托管的服务 |
| 清算结算 | Clearing and Settlement | 票据交易完成后的资金清算和票据结算 |
| DVP | Delivery Versus Payment | 券款对付，指债券交易达成后，在双方指定的结算日，债券和资金同步进行相对交收并互为条件的一种结算方式 |
| 票据池 | Bill Pool | 企业或银行管理的票据集合 |

### 1.3 供应链票据术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 供应链票据 | Supply Chain Bill | 基于供应链金融场景，为供应链上下游企业提供的票据服务 |
| 核心企业 | Core Enterprise | 供应链中处于核心地位的企业 |
| 供应商 | Supplier | 为核心企业提供货物或服务的企业 |
| 经销商 | Dealer | 销售核心企业产品的企业 |
| 应收账款 | Accounts Receivable | 企业在正常的经营过程中因销售商品、产品、提供劳务等业务，应向购买单位收取的款项 |
| 应付账款 | Accounts Payable | 企业应支付但尚未支付的手续费和佣金 |
| 信用增级 | Credit Enhancement | 通过各种方式和手段，提高债务人信用等级的行为 |
