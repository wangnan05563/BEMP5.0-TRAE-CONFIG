## 二、完整技能清单

### 🛠️ 开发实现

#### 1. bemp-personalized-dev — BEMP 个性化开发

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-personalized-dev/` |
| **触发词** | `需求开发`、`功能编码`、`BUG修复`、`个性化开发` |
| **版本** | 稳定 |

**核心能力**：BEMP 票据系统的增量功能开发与修改，严格遵循项目编码规范与目录结构。覆盖前端（Vue）、后端（Java/Spring Boot）、数据库（MySQL）和 Adapter（消息通道）四大开发领域。

**关键文件**：
- `assets/guides/` — 前端/后端/数据库/Adapter 四大开发指南
- `assets/templates/` — Controller、Service、Dto 等标准化代码模板
- `assets/templates/sql/` — DDL/DML/菜单/参数/流程 等 SQL 模板

#### 2. bemp-advanced-doc-generator — 高级文档生成器

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-advanced-doc-generator/` |
| **触发词** | `生成详细设计`、`生成测试用例`、`生成测试报告`、`SIT测试用例` |
| **版本** | v5.0 |

**核心能力**：自动生成详细设计文档、测试用例（Word/Markdown）、Excel SIT 测试用例、测试报告，支持从需求文档自动提取测试点。

**关键文件**：

- `lib/excel-testcase-generator.js` — Excel 测试用例生成器
- `lib/requirement-analyzer.js` — 需求文档智能分析

### 🔍 质量保障

#### 3. bemp-frontend-code-review — 前端代码审查

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-frontend-code-review/` |
| **触发词** | `前端代码走查`、`前端审查`、`前端review` |

**核心能力**：审查 BEMP 各银行个性化前端代码，自动化脚本一键扫描 + 人工逐项审查双模式。支持 12 家银行配置切换。

**自动化检查项**：
- 硬编码中文检测（`check-hardcode.js`）
- 路由注册完整性（`check-routes.js`）
- 国际化覆盖率（`check-i18n.js`）

**报告分级**：🔴阻塞 / 🟠严重 / 🟡警告 / 🟢提示

---

#### 4. bemp-backend-code-review — 后端代码审查

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-backend-code-review/` |
| **触发词** | `后端代码走查`、`后端审查`、`后端review` |
| **版本** | v2.2.0 |

**核心能力**：审查 BEMP 后端代码是否符合项目规范，含代码结构、注解、参数传递、安全性、性能检查。支持多银行配置切换。

**关键文件**：
- `scripts/auto-scan.ps1` — 自动化扫描脚本
- `config/bank-config.json` — 银行配置（包名、前缀、路径）

#### 5. bemp-sonarqube-mcp — SonarQube 代码质量扫描

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-sonarqube-mcp/` |
| **触发词** | `sonar扫描`、`质量门禁`、`代码质量`、`quality gate` |

**核心能力**：基于 SonarQube MCP 的质量门禁检查、新增代码问题扫描、代码片段分析、问题分类与修复建议。

**工作流**：服务器检测启动 → 连接验证 → 代码定位 → 质量门禁检查 → 问题扫描 → 分类报告 → 修复建议 → 修复验证

---

#### 6. bemp-webapp-testing — Web 端自动化测试（首轮）

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-webapp-testing/` |
| **触发词** | `功能测试`、`自动化测试`、`页面验证`、`用例编写` |

**核心能力**：基于 Playwright 的 Web 端自动化测试，覆盖服务健康检查、登录态管理、页面功能验证、控制台错误检测。支持多银行环境无缝切换，306+ 测试场景。

**关键特性**：
- 🏦 多银行配置驱动（`test_config.json`）
- 🔐 会话复用（30 分钟自动续期）
- 📋 用例↔脚本双向索引（`test-index.json`）
- 📊 覆盖 6 个子系统 285+ 页面

**关键文件**：
- `scripts/run_test.py` — 通用测试运行器
- `scripts/login_manager.py` — 登录会话管理
- `scripts/health_check.py` — 服务健康检查

### 🚀 运维部署

#### 8. bemp-automation-startserver — 开发环境启动

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-automation-startserver/` |
| **触发词** | `启动服务`、`重启环境`、`启动Redis`、`启动前端` |

**核心能力**：在 IDE 终端中启动 BEMP 开发环境全栈服务。

**服务清单**：

| 服务 | 端口 |
|------|------|
| Redis | 6379 |
| ZooKeeper | 2181 |
| SpringBoot 后端 | 8010 |
| 前端开发服务器 | 8091 |

**启动顺序**：Redis → ZooKeeper → SpringBoot → Frontend（各服务独立终端）

**关键文件**：
- `scripts/start-bemp-env.ps1` — 环境启动脚本

---

#### 9. bemp-git-maven-automation — Git 同步与 Maven 构建

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-git-maven-automation/` |
| **触发词** | `同步代码`、`编译项目`、`Maven构建`、`Git拉取` |

**核心能力**：自动同步工作空间 Git 仓库并执行 Maven 构建，支持全量/增量编译。可指定 Banks 子目录、构建顺序、并行构建等。

**关键配置**：
- `config/config.properties` — 构建类型、Banks目录、Maven参数、构建顺序

**构建顺序**：bom → framework → adapter → banks → served

---

#### 10. bemp-jenkins-deploy — Jenkins 自动化部署

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-jenkins-deploy/` |
| **触发词** | `Jenkins部署`、`CI/CD`、`自动化部署`、`DevOps` |

**核心能力**：生成前后端分离的 Jenkins Pipeline 自动化部署脚本。支持阶段独立跳过、自动备份与失败回滚、SonarQube 质量门禁、统一健康检查。

**关键特性**：
- 前后端分离部署（Maven + Node.js）
- 自动备份与失败自动回滚
- Redis & Zookeeper 并行启动
- 统一健康检查（TCP/HTTP/Nginx 三模式）

**关键文件**：
- `assets/Jenkinsfile-served` — 后端 Jenkinsfile
- `assets/Jenkinsfile-frontend` — 前端 Jenkinsfile
- `scripts/health-check.ps1` — 健康检查脚本

---

#### 11. bemp-db-operator — 数据库标准化操作

| 属性 | 值 |
|------|-----|
| **目录** | `bemp-db-operator/` |
| **触发词** | `数据库操作`、`执行SQL`、`数据查询`、`数据回退` |
| **版本** | v2.3.1 |

**核心能力**：通过 MCP 实现 Oracle/MySQL 双引擎数据库标准化操作，提供 SQL 执行 → 结果复核 → 异常回退的完整闭环。MySQL 支持事务包裹的安全执行模式。

**关键特性**：
- 双引擎支持（Oracle + MySQL）
- MySQL 安全执行模式（事务包裹）
- 编码预处理（解决中文乱码）
- 版本兼容自动检测
- 批量脚本顺序执行

**关键文件**：
- `scripts/execute-mysql-sql.ps1` — MySQL 执行脚本
- `scripts/execute-oracle-sql.ps1` — Oracle 执行脚本

### ⚡ 性能测试

#### 12. jmeter-performance-test — JMeter 性能测试

| 属性 | 值 |
|------|-----|
| **目录** | `jmeter-performance-test/` |
| **触发词** | `性能测试`、`压力测试`、`JMeter`、`并发测试`、`benchmark` |
