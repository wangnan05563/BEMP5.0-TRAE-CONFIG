## 三、页面之间的关系图

### 系统级关系

```
登录页面 (#/login)
    |
    v
首页仪表盘 (#/mainIndex)
    |
    +---> 系统管理子系统 (#/sm/*)
    +---> 业务管理子系统 (#/bm/*)
    +---> 场内交易子系统 (#/be/*)
    +---> 场外交易子系统 (#/ce/*)
    +---> 票据池子系统 (#/pl/*)
```

### 路由前缀与子系统映射

| 路由前缀 | 子系统 |
|----------|--------|
| `#/sm/` | 系统管理子系统 |
| `#/bm/` | 业务管理子系统 |
| `#/be/` | 场内交易子系统 |
| `#/ce/` | 场外交易子系统 |
| `#/pl/` | 票据池子系统 |
| `#/shcpe/` | CPES管理相关 |

### 业务流程依赖关系

```
[客户管理] ---> [电票签约管理] ---> [承兑业务] ---> [贴现业务]
                                        |               |
                                        v               v
                                   [到期扣款]       [质押业务]
                                   [银票解付]       [托收业务]
                                        |               |
                                        v               v
                                   [场内交易] <--- [追索业务]
                                   [市场交易]      [保证业务]
                                        |
                                        v
                                   [清算结算] ---> [对账管理]
                                        |
                                        v
                                   [账务管理] ---> [流水/分录/科目管理]
```

### 复核流程关系

```
[业务申请] ---> [复核操作] ---> [记账操作]
     |                              |
     v                              v
[跟踪查询] <--- [签收/确认] <--- [交易确认]
```
