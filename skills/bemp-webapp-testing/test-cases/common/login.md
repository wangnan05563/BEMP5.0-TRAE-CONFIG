# 登录流程测试用例

## TC-COMMON-001: 系统登录验证

```yaml
用例编号: TC-COMMON-001
用例名称: 系统登录验证
优先级: P1
前置条件:
  - 后端服务已启动(配置中的端口)
  - 前端服务已启动(配置中的端口)
测试步骤:
  - 步骤1: 导航至登录页面 /#/login
  - 步骤2: 等待登录页面加载完成
  - 步骤3: 输入配置中default角色的用户名
  - 步骤4: 输入配置中default角色的密码
  - 步骤5: 点击"登录"按钮
  - 步骤6: 处理强制登录确认弹窗（点击"是"）
  - 步骤7: 等待首页加载完成(networkidle)
  - 步骤8: 验证登录成功（无"会话已失效"提示）
预期结果:
  - 登录流程正常完成
  - 首页正常显示
  - 无JS错误

测试状态: [待填写]
```
