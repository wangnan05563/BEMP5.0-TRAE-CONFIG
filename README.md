# BEMP 5.0 Trae IDE 配置

这是河南农信 BEMP 票据交易管理平台 V5.0 的 Trae IDE 配置文件和技能集合。

## 目录结构

```
.trae/
├── documents/          # BEMP 开发规范文档
├── learnings/          # 学习记录
├── proactive/          # 主动代理配置
├── rules/              # 项目规则
├── scripts/            # 脚本文件
├── skills/             # Trae 技能
└── specs/              # 需求规格说明
```

## 核心技能

- **bemp-personalized-dev**: 个性化功能开发
- **bemp-automation-startserver**: 开发环境启动
- **bemp-jenkins-deploy**: Jenkins 自动化部署
- **bemp-advanced-doc-generator**: 文档自动生成
- **bemp-backend-code-review**: 后端代码审查
- **bemp-frontend-code-review**: 前端代码审查
- **jmeter-performance-test**: JMeter 性能测试

## 使用方式

1. 将此目录克隆到 BEMP 项目的 `.trae` 目录
2. 在 Trae IDE 中打开项目
3. 使用 `/plan` 或 `/spec` 命令触发技能

## 开发规范

详见 `rules/bemprule.md` 和 `documents/bemp 开发规范/` 目录。
